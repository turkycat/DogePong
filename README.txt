This pong game was written in XNA and uses the Kinect library

To open: You must have the following items:
- Visual Studio 2010 or above with XNA
- Kinect SDK installed

To play using Kinect:
- The kinect automatically detects up to two players. If two-player mode is selected when the Kinect is available, main controls will still be setup using Controllers, but paddles can be controlled with hands.
